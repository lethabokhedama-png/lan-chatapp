# Feature Flags

| Flag | Default | Description |
|------|---------|-------------|
| smart_replies | true | Reply suggestions |
| voice_notes | true | Voice recording |
| disappearing_photos | true | 1x/2x view photos |
| read_receipts | true | Sent/Delivered/Read |
| typing_indicators | true | Typing animation |
| online_presence | true | Online status dots |
| group_mentions | true | @mentions in groups |
| registration_open | true | Allow new signups |
| allow_self_dm | false | Message yourself |
| allow_file_uploads | true | File attachments |
| allow_image_uploads | true | Image sharing |
| allow_group_creation | true | Users can create groups |
| require_approval | false | Admin approves new accounts |
| maintenance_mode | false | Block all logins |
| max_voice_seconds | 300 | Max voice note length |
| max_upload_mb | 10 | Max file size (MB) |
| max_group_members | 50 | Max group size |
| max_message_length | 2000 | Max message characters |
| max_rooms_per_user | 20 | Max rooms per user |
| motd | "" | Message of the day |
